Is HTML Fun to use?

As this is my first time using HTML through this course, it can feel quite daunting.

A hobby of mine is assembling computers, and at first I did not know many things but as I continued it was really intriguing.
I believe the same thing will happen with HTML, thus I made a list.

What are some things I like about HTML?

  1. It is new
  2. It is fun
  3. It is useful
  4. It sparks creativity
